## Project

Backend for the LAMF5's (Liga AcadÃªmica de Mercado Financeiro da UFV) website using GraphQL, Mongoose and Apollo Server

## Technologies

Implemented technologies:

- [GraphQL](https://graphql.org/)
- [Mongoose](https://mongoosejs.com/)
- [Apollo Server](https://www.apollographql.com)

## Build

Install node dependencies:

```bash
npm install
```

Configure the database with environment variables (.env):

```
DB_URI = $prod_url      // Database URI
DB_PORT = $port_number  // API's port
```

Start the backend:

```bash
npm start
```
